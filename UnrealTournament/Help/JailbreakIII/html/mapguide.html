Mapping Guide

 - Information

 

 

 

 

 

 

The mapping guide is not included with the Jailbreak distribution. It can, however, be found at http://www.planetjailbreak.com/resources/mapping/.

 

It contains information about all of the actors, events, properties, and other Jailbreak-specific elements of Jailbreak maps. Any new or existing mappers should read through its entirety before proceeding to create a Jailbreak map.
